# Dirty Metagenomics
## Description

These are a set of scripts that automatizes several procedures via cli's Rscript, to analize converted BIOM files.
### Supported BIOM files:
* Qiime 1.9.x
* Qiime 2 (2018.11 - x)
* BIOM suite (Probably)
